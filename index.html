Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The HyperText Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser.

CSS

  • CSS stands for Cascading Style Sheets
  • This style sheet language is used for defining the presentation of a document written in some markup language like HTML
  • CSS has now become a very important part of building blogs and websites

Git

  • Git is such an essential part of my development workflows
  • Fetch is a command which gets data about remote branches
  • The .git directory, directly where you initialized your repository contains all your local Git data

Javascript

  • A programming language that is one of the core technologies of the World Wide Web, alongside HTML and CSS
  • Where HTML and CSS give structure and style to web pages, JavaScript is responsible for interactive elements that engage the user
  • JavaScript is also used to develop browser-based games
